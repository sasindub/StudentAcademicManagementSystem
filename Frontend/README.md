# Student Academic Management System - Frontend

React.js frontend with Tailwind CSS for the Student Academic Management System.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

3. Open http://localhost:3000 in your browser

## Build for Production

```bash
npm run build
```
